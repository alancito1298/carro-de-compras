Carrito de compras proyecto

El carrito de compras permite a los usuarios agregar productos, modificar la cantidad de productos en el carrito y eliminar productos del carrito. El proyecto utiliza una base de datos MySQL para almacenar la información de los productos y los clientes. El proyecto se implementó en Java 17 y utiliza Spring Boot para el desarrollo de la capa de presentación, el acceso a datos, seguridad y el monitoreo de la aplicación.
